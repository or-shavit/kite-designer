# Kite Color Designer — Requirements

## Overview
A web app that helps kite makers design the color scheme of a specific delta kite. Users pick 3 colors, see a live preview on the kite, and can ask AI to suggest harmonious palettes.

---

## The Kite Design

### Shape
- Delta kite: a large isosceles triangle, point at top, wide base at bottom

### Pattern
The kite has a repeating interlocked weave/ribbon pattern inside it. The pattern creates:

- **Band A** — one set of diagonal ribbon strips
- **Band B** — a second set of diagonal ribbon strips (different direction)
- **Band C** — a third set of diagonal ribbon strips + the outer border of the kite
- **Inner triangles** — small triangles at the center of each weave cell, always **white** (fixed, not user-controlled)

Reference images:
- `kite-design.jpeg` — the black-and-white sketch of the kite pattern
- `kite-example.jpeg` — a colored example showing black/red/yellow bands

### Pattern Structure
- 4 rows of weave cells arranged in a triangle (1 cell in row 1, 2 in row 2, 3 in row 3, 4 in row 4 = 10 upward + 6 downward = 16 cells total)
- Each cell is subdivided into 4 panels: 3 colored corners (A, B, C) + 1 white center triangle
- Total: ~64 colored panels + outer border

## Features

### 1. Live Kite Preview
- Large SVG of the kite displayed prominently
- Updates in real-time as the user changes colors
- Accurate to the reference design

### 2. Color Pickers
- 3 color pickers, one per band (A, B, C)
- Each shows the current hex value
- Changes update the kite preview instantly

### 3. AI Palette Suggestions (Claude API)
- A text field where the user can optionally describe a preference (e.g. "warm sunset tones", "bold and contrasting")
- A "Suggest Colors" button that calls Claude Haiku
- Returns 3 palette suggestions, each with 3 hex colors and a label
- Clicking a suggested palette applies all 3 colors to the kite

### 4. Design Name
- The current color combination is automatically given a descriptive name (e.g. "Desert Sunset", "Arctic Storm", "Tropical Canopy")
- The name updates whenever the colors change (debounced — not on every keystroke)
- Name is generated by Claude Haiku based on the 3 hex colors
- Displayed prominently above or below the kite

### 5. Share Button
- Encodes the current 3 colors into the URL as query params (e.g. `?a=%23FF0000&b=%23000000&c=%23FFD700`)
- On page load, reads colors from URL params if present (so shared links restore the exact design)
- Share button uses Web Share API on mobile, falls back to copying the URL to clipboard on desktop
- Shows "Copied!" confirmation briefly after copying

### 6. Reset
- A button to restore the default colors

## Technical Stack
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **AI**: Claude Haiku (`claude-haiku-4-5-20251001`) via `@anthropic-ai/sdk`
- **Rendering**: Inline SVG with React

---

## Design Style
- Light theme resembling an open sky
- Background: soft sky blue gradient (e.g. `#E0F0FF` to `#F5FAFF`)
- Cards/panels: white or very light blue-white (`#FFFFFF` / `#F0F8FF`)
- Text: deep navy or slate (`#1A2E4A`)
- Accent color: sky blue (`#4A9FD4`) for buttons and highlights
- Clean, airy look suited for kite makers
- Kite SVG should be the visual centerpiece

---

## Pages / Routes
- `/` — main page with kite preview + color controls
- `/api/suggest` — POST endpoint, calls Claude, returns palette suggestions
- `/api/name` — POST endpoint, calls Claude, returns a design name for the given 3 colors

---

## Out of Scope
- Saving/exporting designs
- User accounts
- Multiple kite shapes
- Changing the white inner triangles
- Mobile-specific optimizations (responsive is nice-to-have)

---

## Example User Flow
1. User opens the app → sees kite with default colors + auto-generated name
2. User adjusts colors → kite updates live, name refreshes after a short pause
3. User types "ocean vibes" → clicks "Suggest Colors" → 3 palettes appear → clicks one → kite updates
4. User clicks "Share" → URL is copied → sends link to a colleague
5. Colleague opens link → sees exact same kite design with the same name
